Run Scrapy spiders from a terminal. The DjangoPipeline saves items directly into Django models (MCQ + PastPaper). No browser required — Scrapy handles concurrency and retries.

# Run from: c:\laragon\www\imtihanBackend\
# TestPointPK — PPSC, first 10 papers
scrapy crawl testpoint -s EXAM=ppsc -s MAX_PAPERS=10 --project scraper
# TestPointPK — FPSC English only
scrapy crawl testpoint -s EXAM=fpsc -s SUBJECT_KW=english --project scraper
# GoTest.pk — All subjects
scrapy crawl gotest --project scraper
# GoTest.pk — English only, max 10 tests
scrapy crawl gotest -s SUBJECT_KW=english -s MAX_TESTS=10 --project scraper
# PastPapersPDF — PMS, max 20 posts
scrapy crawl pastpapers -s EXAM=pms -s MAX_POSTS=20 --project scraper
# PastPapersPDF — PPSC GK subject filter
scrapy crawl pastpapers -s EXAM=ppsc -s SUBJECT_KW=gk --project scraper
# All exams, all posts
scrapy crawl pastpapers --project scraper
Tip: Set DJANGO_SETTINGS_MODULE=imtihanBackend.settings in your environment before running Scrapy so the pipeline can access Django models.
